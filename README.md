# Todo App for Android

## Overview
This project is a simple Todo application designed for the Android platform. It is built using Android Studio and follows best practices for Android development.

## Features
- **Add Tasks**: Users can add new tasks to their todo list.
- **Delete Tasks**: Users can remove tasks from their list.
- **Edit Tasks**: Users can modify existing tasks.
- **Task Persistence**: The app stores tasks locally using Room database.
- **User-friendly Interface**: The UI is designed to be intuitive and easy to navigate.

## Tech Stack
- **Kotlin**: The primary programming language used for the app.
- **Android Jetpack**: Utilizes components such as ViewModel and LiveData.
- **Room**: For database management and data persistence.

## Getting Started
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/jordanforesta33-gif/todo-app-android.git
   ```
2. **Open in Android Studio**.
3. **Build the Project**: Ensure all dependencies are downloaded and the project builds successfully.

## Usage
- Once the app is launched, you can start adding your tasks. 
- Tasks can be marked as completed or deleted as needed.
